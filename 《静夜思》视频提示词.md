
## 风格规范
1. 色彩体系：青灰主色调搭配月白银边
2. 动态效果：水墨晕染过渡+粒子光尘
3. 节奏把控：诗句间隔保留0.5秒气口
4. 音效设计：竹简碰撞声+远山风铃
